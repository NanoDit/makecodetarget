# NanoDit MakeCode

Welcome to the NanoDit MakeCode editor — a block and Python coding environment designed for the NanoDit ESP32‑S3 board. The editor provides a simple, visual way to create programs using drag‑and‑drop blocks or typed Python code.

---

## What you can do with NanoDit

NanoDit includes a 5×5 LED matrix, buttons, sensors, sound, and wireless communication. With MakeCode, you can:

- Light up the LED display
- Detect button presses and motion
- Play tones and melodies
- Read sensor values
- Send and receive radio messages
- Build interactive projects with blocks or Python

---

## Block Categories

NanoDit supports the same categories you may know from the micro:bit editor:

- **Basic** — show text, pause, loops
- **Input** — buttons, shake detection
- **Music** — tones and melodies
- **LED** — control the 5×5 display
- **Radio** — send and receive messages
- **Loops** — repeat actions
- **Logic** — conditions and comparisons
- **Variables** — store values
- **Math** — calculations and random numbers

All categories use the built‑in MakeCode icons and colors.

---

## Python Support

Switch to Python at any time to view or edit the code behind your blocks. NanoDit uses a MicroPython‑based runtime, so Python programs run directly on the board.

---

## Simulator

The built‑in simulator lets you test your code without hardware. It includes:

- A virtual 5×5 LED matrix  
- Buttons A and B  
- A speaker indicator  
- Radio activity  
- Motion simulation  

---

## Getting Started

1. Open a new project  
2. Drag blocks from the toolbox  
3. Test your program in the simulator  
4. Connect your NanoDit board  
5. Download your program to the device
